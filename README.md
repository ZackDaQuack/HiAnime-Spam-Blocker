# HiAnime-Spam-Blocker

Userscript that hides junk from HiAnime comments because the devs won't do it. (they definitely are not getting paid off)

## Installation

1. Install **Violentmonkey** (Chrome/Edge/Firefox/Etc): https://violentmonkey.github.io
2. Click this link:
